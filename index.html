Introdução

Esta página apresenta um resumo das principais tags HTML aprendidas no bootcamp Santander 2025 - Front-end, oferecido pela plataforma DIO. Aqui você encontrará explicações sobre tags utilizadas em formulários, elementos para inserção de mídias e também as tags essenciais para a criação de tabelas.

Formulários

As principais tags utilizadas para a criação de formulários em HTML são:

  • <form>: Define o início e o fim de um formulário. É o elemento principal que agrupa todos os campos e botões.
  • <input>: Utilizada para criar campos de entrada de dados, como texto, senha, e-mail, número, entre outros. O tipo de entrada é definido pelo atributo type.
  • <label>: Associa um texto descritivo a um campo do formulário, melhorando a acessibilidade.
  • <select>: Cria uma lista suspensa de opções para o usuário escolher.
  • <option>: Define cada uma das opções dentro de um <select>.
  • <textarea>: Permite a inserção de textos longos em várias linhas.
  • <button>: Cria botões para enviar ou redefinir os dados do formulário.
  • <input type="radio">: Permite a seleção de apenas uma opção entre várias disponíveis.
  • <input type="checkbox">: Permite a seleção de múltiplas opções independentes.

Mídias

As tags HTML para mídias permitem incorporar imagens, sons, vídeos e conteúdos externos em páginas web. A seguir, estão as principais tags utilizadas:

  • <img>: Utilizada para exibir imagens. O atributo src define o caminho do arquivo, enquanto alt fornece uma descrição alternativa. Os formatos mais comuns são .jpg, .png, .gif e .svg.
  • <audio>: Permite inserir arquivos de áudio. Os principais atributos são controls (exibe controles de reprodução), autoplay (inicia automaticamente), loop (reproduz em loop) e muted (inicia sem som). Os formatos suportados incluem .mp3, .wav e .ogg.
  • <video>: Usada para incorporar vídeos. Possui atributos como controls, autoplay, loop, muted e poster (imagem de capa). Os formatos mais comuns são .mp4, .webm e .ogg.
  • <iframe>: Permite incorporar conteúdos externos, como vídeos do YouTube, mapas ou outras páginas web, através do atributo src.
  • <source>: Usada dentro das tags <audio> e <video> para especificar múltiplos arquivos de diferentes formatos, garantindo compatibilidade entre navegadores. O navegador escolhe o primeiro formato suportado.
  • <track>: Utilizada dentro da tag <video> para adicionar faixas de texto, como legendas, descrições ou capítulos, melhorando a acessibilidade.

O encadeamento de recursos ocorre quando múltiplos arquivos são especificados usando a tag <source> dentro de <audio> ou <video>. Dessa forma, o navegador seleciona automaticamente o formato mais adequado, garantindo que o conteúdo seja exibido corretamente para todos os usuários.

Tabelas

As tabelas em HTML são criadas para organizar dados em linhas e colunas, facilitando a visualização de informações estruturadas. As principais tags utilizadas para a construção de tabelas são:

  • <table>: Define o início e o fim de uma tabela.
  • <tr>: Representa uma linha dentro da tabela.
  • <td>: Define uma célula de dados em uma linha.
  • <th>: Cria uma célula de cabeçalho, geralmente utilizada para títulos das colunas ou linhas.
  • <thead>: Agrupa o cabeçalho da tabela.
  • <tbody>: Agrupa o corpo principal da tabela.
  • <tfoot>: Agrupa o rodapé da tabela.
  • <caption>: Adiciona um título descritivo à tabela.

Os principais atributos das tags de tabela incluem:

  • colspan: Permite que uma célula ocupe várias colunas.
  • rowspan: Permite que uma célula ocupe várias linhas.
  • scope: Usado em <th> para indicar se o cabeçalho se refere a uma linha ou coluna (row ou col).

Essas tags e atributos tornam possível criar tabelas acessíveis, organizadas e personalizadas conforme a necessidade do conteúdo apresentado.

Exemplo de tabela:

<table>
                    <caption>Exemplo de Tabela de Participantes</caption>
                    <thead>
                        <tr>
                            <th scope="col">Nome</th>
                            <th scope="col">Função</th>
                            <th scope="col">Área</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td>Ana</td>
                            <td>Mentora</td>
                            <td rowspan="2">Front-end</td>
                        </tr>
                        <tr>
                            <td>Carlos</td>
                            <td>Aluno</td>
                        </tr>
                        <tr>
                            <td>Beatriz</td>
                            <td colspan="2">Full Stack</td>
                        </tr>
                    </tbody>
                    <tfoot>
                        <tr>
                            <td colspan="3">Total de participantes: 3</td>
                        </tr>
                    </tfoot>
                </table>
